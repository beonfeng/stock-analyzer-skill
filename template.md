# Stock Analyzer 报告模板说明

## 模板文件

完整报告模板位于 `templates/report-template.md`

## 使用方法

1. 运行 `analyze_stock.py` 脚本生成报告
2. 脚本会自动填充所有数据
3. 报告保存在 `股票代码-股票名称/` 目录下

## 报告章节

| 章节 | 内容 |
|------|------|
| 总结 | 综合评级、关键指标、风险提示 |
| 一、行情概览 | 当日价格、涨跌幅、成交量 |
| 二、技术分析 | 均线、MACD、KDJ、RSI、布林带 |
| 三、资金分析 | 主力资金流向、北向资金 |
| 四、基本面分析 | 估值指标、盈利与成长 |
| 五、财务排雷 | 风险指标检查和排雷结论 |
| 六、新闻动态 | 近期新闻 |
| 七、行业板块排名 | 当日行业涨跌排名 |
| 八、反证清单与跟踪因子 | 需要跟踪的关键指标 |
| 风险提示 | 免责声明 |

## 自定义模板

如需自定义报告格式，修改 `analyze_stock.py` 中的 `generate_report` 函数。
