# 数据源说明

## 概述

Stock Analyzer 直连东方财富 API，绕过系统代理，自动重试。所有数据均来自公开渠道。

## 数据源列表

### K 线数据

- **API 地址**：`push2his.eastmoney.com/api/qt/stock/kline/get`
- **数据内容**：开盘价、收盘价、最高价、最低价、成交量、成交额、振幅、涨跌幅、涨跌额、换手率
- **时间范围**：最多 500 个交易日
- **更新频率**：实时

### 实时行情

- **API 地址**：
  - `push2.eastmoney.com/api/qt/clist/get`（从列表查询）
  - `push2.eastmoney.com/api/qt/stock/get`（单只查询）
- **数据内容**：最新价、涨跌幅、市盈率、市净率、总市值、流通市值、ROE、毛利率、营收同比、净利润同比、资产负债率
- **更新频率**：实时

### 资金流向（个股分层）

- **API 地址**：`push2his.eastmoney.com/api/qt/stock/fflow/daykline/get`
- **数据内容**：逐日资金流向，完整订单分层（超大单/大单/中单/小单净流入+占比）
- **注意**：`push2.eastmoney.com` 的 f62/f66-f87 字段不可靠（f62 始终为 1），已弃用
- **更新频率**：每日盘后更新

### 逐笔成交明细 [v1.3.0]

- **API 地址**：`push2.eastmoney.com/api/qt/stock/details/get`
- **数据内容**：当日全部逐笔成交（时间、价格、手数、成交笔数）
- **用途**：大单异动检测、主力聚集识别
- **更新频率**：实时

### 板块资金全景 [v1.3.0]

- **API 地址**：`push2.eastmoney.com/api/qt/clist/get`（`fs=m:90+t:2`）
- **数据内容**：全市场 100+ 行业板块资金流向排名
- **更新频率**：实时

### 北向资金

- **API 地址**：`push2his.eastmoney.com/api/qt/stock/kline/get`
- **数据内容**：沪股通、深股通每日资金流向
- **更新频率**：每日

### 资讯数据（研报 + 公告）

- **研报 API**：`reportapi.eastmoney.com/report/list`（个股研究报告，含机构评级）
- **公告 API**：`np-anotice-stock.eastmoney.com/api/security/ann`（公司公告）
- **数据内容**：研报标题、发布机构、发布日期 / 公告标题、公告日期
- **更新频率**：实时
- **说明**：原新闻 API（np-listapi.eastmoney.com）已下线，使用研报+公告替代

### 行业数据

- **API 地址**：`push2.eastmoney.com/api/qt/clist/get`
- **数据内容**：行业板块涨跌幅、总市值、换手率
- **更新频率**：实时

### 财务报表

- **API 地址**：`datacenter.eastmoney.com/api/data/get`
- **类型参数**：`RPT_DMSK_FN_INCOME`（利润表）、`RPT_DMSK_FN_CASHFLOW`（现金流量表）
- **数据内容**：现金流量表、资产负债表关键指标
- **更新频率**：季度

### 历史财务趋势 [v1.3.0]

- **API 地址**：同财务报表（近 5 年季度数据）
- **数据内容**：营收/净利润/营业成本/经营现金流 5 年趋势
- **用途**：营收 CAGR、利润 CAGR、净利率走势、现金流质量评估

## 数据源可信度

数据可信度默认排序：

1. 公司公告、年报、季报、交易所文件
2. 投资者关系活动记录、业绩说明会、交易所互动平台
3. 东方财富、新浪财经、同花顺等结构化数据
4. 券商研报、行业报告、协会数据、统计局数据
5. 新闻媒体、社交媒体、市场传闻

## API 限制

- **请求频率**：自动重试机制，最多 8 次
- **超时时间**：15 秒
- **代理**：绕过系统代理，直连 API

## 数据处理

### 价格处理

- 价格类字段（f43, f162, f167, f170）：需要除以 100
- 例如：8250 → 82.50

### 比率处理

- 比率类字段（f173, f185, f186, f188）：已经是百分比小数形式，直接使用
- 例如：5.56 → 5.56%

### 金额处理

- 金额类字段（f116, f117）：已经是元，直接使用
- 例如：627999000000 → 6279.99 亿

## 错误处理

### 网络错误

- 自动重试最多 8 次
- 每次重试间隔递增（2秒 + 0.5秒 × 尝试次数）
- 最终失败返回空数据或默认值

### 数据异常

- 缺失字段返回默认值 0
- 非数字值安全转换为 0
- 空字符串返回 "-"

## 股票名称映射

当 API 获取名称失败时，使用内置映射表：

```python
_STOCK_NAMES = {
    "000001": "平安银行",
    "000002": "万科A",
    "000333": "美的集团",
    "600519": "贵州茅台",
    # ... 更多股票
}
```

## 使用建议

1. **数据验证**：重要决策前，建议与公司公告核对关键数据
2. **实时性**：盘中数据实时更新，盘后数据为当日收盘数据
3. **历史数据**：最多获取 500 个交易日，如需更长时间段需自行存储
4. **批量分析**：建议控制批量股票数量，避免请求过于频繁
