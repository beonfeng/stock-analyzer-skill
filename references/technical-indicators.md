# 技术指标说明

## 均线（MA）

均线（Moving Average）是将过去 N 天的收盘价取平均值连成的曲线。

### 常用周期

| 周期 | 用途 | 说明 |
|------|------|------|
| MA5 | 短期趋势 | 一周（5个交易日）平均成本 |
| MA10 | 短期趋势 | 两周平均成本 |
| MA20 | 中期趋势 | 一个月平均成本 |
| MA60 | 中期趋势 | 一个季度平均成本 |
| MA120 | 长期趋势 | 半年平均成本 |
| MA250 | 长期趋势 | 一年平均成本（年线） |

### 信号判断

- **多头排列**：MA5 > MA10 > MA20 > MA60，趋势向上
- **空头排列**：MA5 < MA10 < MA20 < MA60，趋势向下
- **金叉**：短期均线上穿长期均线，买入信号
- **死叉**：短期均线下穿长期均线，卖出信号
- **支撑/阻力**：均线可作为支撑位或阻力位

## MACD

MACD（Moving Average Convergence Divergence，指数平滑异同移动平均线）由 DIF 线和 DEA 线组成。

### 计算方法

1. DIF = EMA(12) - EMA(26)  （短期EMA减去长期EMA）
2. DEA = EMA(DIF, 9)  （DIF的9日EMA）
3. MACD柱 = (DIF - DEA) × 2

### 信号判断

- **金叉**：DIF 上穿 DEA，买入信号
- **死叉**：DIF 下穿 DEA，卖出信号
- **MACD柱由负转正**：多头增强
- **MACD柱由正转负**：空头增强
- **顶背离**：股价新高，MACD不创新高，看跌
- **底背离**：股价新低，MACD不创新低，看涨

## KDJ

KDJ（随机指标）用于判断超买超卖状态，取值范围 0~100。

### 计算方法

1. RSV = (收盘价 - 最低价) / (最高价 - 最低价) × 100
2. K = EMA(RSV, 3)
3. D = EMA(K, 3)
4. J = 3K - 2D

### 信号判断

- **K > 80**：超买区，价格偏高，可能回调
- **K < 20**：超卖区，价格偏低，可能反弹
- **J > 100**：极度超买
- **J < 0**：极度超卖
- **金叉**：K 上穿 D，买入信号
- **死叉**：K 下穿 D，卖出信号

## RSI

RSI（Relative Strength Index，相对强弱指数）衡量一段时间内涨幅与跌幅的比值，取值 0~100。

### 计算方法

1. RS = 平均涨幅 / 平均跌幅
2. RSI = 100 - 100 / (1 + RS)

### 常用周期

| 周期 | 用途 |
|------|------|
| RSI6 | 短期超买超卖 |
| RSI12 | 中期超买超卖 |
| RSI24 | 长期超买超卖 |

### 信号判断

- **> 80**：超买，短期可能见顶
- **< 20**：超卖，短期可能见底
- **> 50**：多方占优
- **< 50**：空方占优
- **顶背离**：股价新高，RSI不创新高，看跌
- **底背离**：股价新低，RSI不创新低，看涨

## 布林带（BOLL）

布林带（Bollinger Bands）由三条轨道组成。

### 计算方法

1. 中轨 = MA(20)  （20日均线）
2. 上轨 = 中轨 + 2 × 标准差(20)
3. 下轨 = 中轨 - 2 × 标准差(20)

### 信号判断

- **价格触及上轨**：可能超买回调
- **价格触及下轨**：可能超卖反弹
- **带宽收窄**：即将变盘
- **带宽扩大**：趋势延续
- **价格突破上轨**：强势上涨
- **价格跌破下轨**：弱势下跌

## ATR

ATR（Average True Range，平均真实波幅）衡量价格波动的剧烈程度。

### 计算方法

1. TR = MAX(最高价-最低价, |最高价-前收盘价|, |最低价-前收盘价|)
2. ATR = MA(TR, 14)  （14日平均）

### 用途

- **衡量波动性**：ATR 越大，波动越剧烈
- **设置止损**：止损价 = 买入价 - N × ATR
- **仓位管理**：仓位 = 风险金额 / (N × ATR)

## 综合判断

### 趋势判断

| 条件 | 判断 |
|------|------|
| 价格 > MA20 > MA60 | 中期多头 |
| 价格 < MA20 < MA60 | 中期空头 |
| MACD金叉 + KDJ金叉 | 强势看涨 |
| MACD死叉 + KDJ死叉 | 强势看跌 |

### 超买超卖判断

| 条件 | 判断 |
|------|------|
| KDJ > 80 + RSI > 70 | 超买，注意回调 |
| KDJ < 20 + RSI < 30 | 超卖，注意反弹 |
| 价格突破布林上轨 | 强势，但注意回调 |
| 价格跌破布林下轨 | 弱势，但注意反弹 |

### 买卖信号

| 信号 | 条件 | 操作 |
|------|------|------|
| 强势买入 | MACD金叉 + KDJ金叉 + RSI < 50 | 可以买入 |
| 弱势卖出 | MACD死叉 + KDJ死叉 + RSI > 50 | 可以卖出 |
| 观望 | 信号不一致 | 等待确认 |

## 扩展技术指标

### RSI 背离

RSI 背离是重要的趋势反转信号：

- **顶背离**：价格创新高，但 RSI 未创新高 → 看跌信号
- **底背离**：价格创新低，但 RSI 未创新低 → 看涨信号

### MACD 柱状图

MACD 柱状图（DIF - DEA）反映多空力量对比：

- 红柱（DIF > DEA）：多头占优
- 绿柱（DIF < DEA）：空头占优

### 成交量异动

| 状态 | 定义 | 信号 |
|------|------|------|
| 放量 | 当日成交量 > 5日均量 × 1.5 | 可能有大资金介入 |
| 缩量 | 当日成交量 < 5日均量 × 0.5 | 市场观望情绪浓厚 |
| 天量 | 当日成交量 > 20日均量 × 2.0 | 往往是变盘信号 |
| 地量 | 当日成交量 < 20日均量 × 0.3 | 可能见底 |

### K 线形态

| 形态 | 识别条件 | 信号 |
|------|----------|------|
| 十字星 | 开盘价 ≈ 收盘价 | 犹豫信号 |
| 锤子线 | 下影线 > 实体 2 倍 | 底部看涨 |
| 吞没形态 | 当日实体包含前一日实体 | 趋势反转 |

### 筹码分布

- **获利盘比例**：当前价格下方的筹码占比
- **套牢盘比例**：当前价格上方的筹码占比
- **筹码集中度**：价格波动越小，筹码越集中
