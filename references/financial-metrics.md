# 财务指标说明

## 估值指标

### 市盈率（PE）

市盈率（Price to Earnings Ratio）= 股价 / 每股收益

| 类型 | 说明 | 用途 |
|------|------|------|
| 静态PE | 股价 / 上年每股收益 | 历史对比 |
| 动态PE | 股价 / 预测每股收益 | 未来预期 |
| TTM PE | 股价 / 过去12个月每股收益 | 最常用 |

**判断标准**：
- PE < 0：亏损状态
- PE 0-20：低估值
- PE 20-40：合理估值
- PE 40-80：高估值
- PE > 80：极高估值，可能泡沫

### 市净率（PB）

市净率（Price to Book Ratio）= 股价 / 每股净资产

**判断标准**：
- PB < 1：破净，股价低于净资产
- PB 1-2：低估值
- PB 2-4：合理估值
- PB > 4：高估值

### 市销率（PS）

市销率（Price to Sales Ratio）= 总市值 / 营业收入

**判断标准**：
- PS < 1：低估值
- PS 1-3：合理估值
- PS > 3：高估值

## 盈利能力指标

### ROE（净资产收益率）

ROE（Return on Equity）= 净利润 / 净资产 × 100%

**判断标准**：
- ROE < 5%：盈利能力弱
- ROE 5-15%：盈利能力一般
- ROE 15-25%：盈利能力强
- ROE > 25%：盈利能力很强

### ROA（总资产收益率）

ROA（Return on Assets）= 净利润 / 总资产 × 100%

**判断标准**：
- ROA < 2%：资产利用效率低
- ROA 2-5%：资产利用效率一般
- ROA > 5%：资产利用效率高

### 毛利率

毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%

**判断标准**：
- 毛利率 < 20%：低毛利行业
- 毛利率 20-40%：中毛利行业
- 毛利率 > 40%：高毛利行业

### 净利率

净利率 = 净利润 / 营业收入 × 100%

**判断标准**：
- 净利率 < 5%：盈利能力弱
- 净利率 5-15%：盈利能力一般
- 净利率 > 15%：盈利能力强

## 成长能力指标

### 营收同比

营收同比 = (本期营业收入 - 上期营业收入) / 上期营业收入 × 100%

**判断标准**：
- 营收同比 < 0：收入下降
- 营收同比 0-10%：低增长
- 营收同比 10-30%：中增长
- 营收同比 > 30%：高增长

### 净利润同比

净利润同比 = (本期净利润 - 上期净利润) / 上期净利润 × 100%

**判断标准**：
- 净利润同比 < 0：利润下降
- 净利润同比 0-10%：低增长
- 净利润同比 10-30%：中增长
- 净利润同比 > 30%：高增长

## 偿债能力指标

### 资产负债率

资产负债率 = 总负债 / 总资产 × 100%

**判断标准**：
- 资产负债率 < 40%：低负债，财务稳健
- 资产负债率 40-60%：中等负债
- 资产负债率 60-70%：高负债
- 资产负债率 > 70%：超高负债，财务风险大

### 流动比率

流动比率 = 流动资产 / 流动负债

**判断标准**：
- 流动比率 < 1：短期偿债能力弱
- 流动比率 1-2：短期偿债能力一般
- 流动比率 > 2：短期偿债能力强

### 速动比率

速动比率 = (流动资产 - 存货) / 流动负债

**判断标准**：
- 速动比率 < 0.5：短期偿债能力弱
- 速动比率 0.5-1：短期偿债能力一般
- 速动比率 > 1：短期偿债能力强

## 营运能力指标

### 存货周转率

存货周转率 = 营业成本 / 平均存货

**判断标准**：
- 存货周转率越高，存货管理效率越高
- 需要与行业均值对比

### 应收账款周转率

应收账款周转率 = 营业收入 / 平均应收账款

**判断标准**：
- 应收账款周转率越高，回款速度越快
- 需要与行业均值对比

### 总资产周转率

总资产周转率 = 营业收入 / 平均总资产

**判断标准**：
- 总资产周转率越高，资产利用效率越高
- 需要与行业均值对比

## 现金流指标

### 经营现金流/净利润

经营现金流/净利润 = 经营活动现金流量净额 / 净利润

**判断标准**：
- > 0.7：利润质量高
- 0.3-0.7：利润质量一般
- < 0.3：利润质量低，可能有应收账款问题

### 经营现金流/营业收入

经营现金流/营业收入 = 经营活动现金流量净额 / 营业收入 × 100%

**判断标准**：
- > 10%：现金流良好
- 5-10%：现金流一般
- < 5%：现金流较差

## 财务排雷规则

### 红灯指标（一票否决）

| 指标 | 红灯条件 | 风险 |
|------|----------|------|
| 资产负债率 | > 70% | 财务风险大 |
| 市盈率 | < 0 | 亏损状态 |
| 净利润同比 | < -30% | 利润大幅下滑 |

### 黄灯指标（预警）

| 指标 | 黄灯条件 | 风险 |
|------|----------|------|
| 资产负债率 | 60-70% | 负债偏高 |
| 市盈率 | > 80 | 估值过高 |
| 净利润同比 | -30% ~ -10% | 利润下滑 |

### 绿灯指标（安全）

| 指标 | 绿灯条件 | 说明 |
|------|----------|------|
| 资产负债率 | < 60% | 负债合理 |
| 市盈率 | 0-80 | 估值合理 |
| 净利润同比 | > 0% | 利润增长 |
